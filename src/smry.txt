[ 중개업소 ]
1. 중개업소 회원가입
  1.1 간편가입 : 기존 리브온 회원
    - 대표자명과 휴대폰 번호로 본인 인증 후, 중개업소 원장에 데이터가 있을경우 간편가입 진행
    - 간편가입 시 입력한 정보를 기존 원장에 update 시킨 후 회원가입 요청( ag01, ag30 ...)
    - 어드민에서 회원가입 승인/재요청 처리 ( Email, Lms, push 발송)

  1.2 신규가입
    - 대표자 이름을 통해 국가공간정보포탈에서 중개업소를 조회 후 본인 중개업소를 선택한다.
    - 휴대폰 본인 인증후 회원가입을 요청한다. ( ag01, ag30 ...)
    - 어드민에서 회원가입 승인/재요청 처리 ( Email, Lms, push 발송)

2. 중개업소 현황

3. 중개업소 조회
  3.1 중개허브 회원
    - 정보승인구분이 '2'인 중개업소중 tslndeg16에 가입요청 후 승인된 중개업소를 조회한다.
  3.2 구 공인중개사전용관 회원
    - 정보승인구분이 '2'인 중개업소중 tslndeg16에 가입요청이 없는 중개업소를 조회한다.
  3.3 중개업소 탈퇴 회원
    - 정보승인구분이 3인 중개업소
  3.4 폐업 중개업소
    - 중개업소 폐업여부가 1 인 중개업소

4. 중개업소 수기등록
    - 휴폐업 조회, 휴대폰번호 중복조회, 중개업소등록번호 중복조회 후 가입 요청
    - tslndag01 중개업소 원장 생성 (수기등록여부:1, 정보승인구분:1)
    - tslndag04 조사희망단지 Insert
    - tslndag30 대표직원 Insert
    - tslndes16 가입신청 Insert ( 요청내용구분: 5 )
    
5. 중개업소 신규관리
  5.1 승인
    - tslndag01 정보승인구분 '2' update
    - tslndes16 정보변경처리상태구분 '3' update
    - tslndag30 use_yn '1'
  5.2 재요청
    - tslndes16 정보변경처리상태구분 '4' update
  - 수기등록 중개업소는 기본정보 수정 가능
  - Email, Lms, push 발송 

6. 중개업소 변경관리
  - 중개업소등록증, 시세조사단지, 대표사진, 회원탈퇴, 대표전화 변경 요청 조회 및 처리
  - 정보승인, 보류 처리.

7. 중개업소 문의관리

[ 매물관리 ]

1. 매물등록
  1.1 자체매물
    - 중개사허브센터에서 매물 등록 > 홍보의뢰서 발송(집주인) > 어드민 공적장부 수기검증 > 매물 노출(1달)
     
    < 배치 >
      - 홍보의뢰서 미인증 시 PUSH발송, (매물 등록 직원대상으로 하루에 1번씩 최대 3번 발송)
      - 홍보의뢰서 10일 미인증 시 자체매물 폐기 및 PUSH발송
      - 30일 노출 매물 폐기

  2.2 제휴매물
    - 배치를 활용해 제휴사에서 매물정보를 받아옴.

2. 홍보의뢰서 확인 / 공적장부 검증
    - 공적장부 수기검증 후 30일 노출 시작.

3. 매물 조회 및 사진&유튜브 관리
 : 등록된 매물조회와 등록된 매물의 사진, 유튜브 링크를 삭제/복원 할 수 있다.
  
  3.1 매물원장 팝업
    - 매물의 상세 정보를 확인할 수 있다.
    - 매물을 일시적으로 노출중지 할 수 있다.
  
  3.2 매물 사진&유튜브 삭제
    - 스팸/음란물/욕설/명예훼손 사유 신고 시 매물 패널티 적용
        ** 자체매물 패널티: ag01 매물사진등록제한시작일, 종료일 update ( 1주일 ) 
        ** 제휴매물 패널티: 해당 중개업소가 등록한 제휴매물의 이미지노출여부 '0' 으로 update
                           eg71(제휴매물 패널티) 삭제 매물 정보와 중개업소 정보 insert ( 배치성으로 eg71에 있는 중개업소의 제휴매물 사진 노출 정지) 
  3.3 매물 사진&유튜브 복원
    - 자체매물: ag01의 매물사진등록제한일을 최근 삭제일시로 업데이트, 없으면 null
    - 제휴매물
        *최근 삭제건이 없는 경우: eg0에서 이미지 노출여부 '1' 업데이트, eg71에서 delete
        *최신 삭제건이 있는 경우: 최근 삭제 매물정보로 eg71 update
    < 배치작업 >
     - 패널티 만료 중개업소 사진 노출 시작 : eg05 이미지노출여부 '1', eg71 delete
     - 패널티 중개업소 매물사진 노출 중지(제휴) : eg05 이미지노출여부 '0'
